- **File Format**: PDF only (ensures consistent display across devices)
- **File Size**: Keep under 2MB for fast downloads
- **Filename**: Keep the exact name `Azhar-Ali-Resume.pdf` or update the download link in `index.html`
- **Content**: Include contact info, skills, experience, education, projects
- **Design**: Match the cyber security theme of the portfolio (optional but recommended)
